## 硬件准备 ##
### 基础 ###
 - 树莓派（关于Arduino的文章会在后续考虑出）
 - 20kg（或以上）扭力舵机**（注意最低电压不要大于5V）**
 - 舵臂
 - 杜邦线若干
 - 不容易断的绳子（我把USB线剪了...）
 - 透明胶
 - 502等强力胶水
 - 盒子

### 进阶 ###
 - 微雪2.9inch墨水屏

## 软件准备 ##
 - Raspbian系统桌面版（最好用NOOBS）
 - itchat库
 - Flask框架
